# mlb-live

MLB scores, results, and headlines Discord bot. Posts daily MLB updates including yesterday's scores and today's schedule to Discord every morning during baseball season.

## Topics
python, discord, bot, mlb, baseball, sports, scores

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/mlb-live.git
cd mlb-live

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 mlb-live.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
